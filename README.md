# Homebrew Tap for yapi

This is the official Homebrew tap for [yapi](https://github.com/jamierpond/yapi).

## Installation

```bash
brew tap jamierpond/yapi
brew install --cask yapi
```

## What is yapi?

yapi is a CLI-first, offline-first, git-friendly API client for HTTP, gRPC, and TCP. It uses simple YAML files to define requests, meaning you can commit them, review them, and run them anywhere.

For more information, visit the [main repository](https://github.com/jamierpond/yapi).
